# CSS Themes

This folder contains different color theme versions for your portfolio.

## Available Themes

1. **main-dark.css** - Dark theme (currently active)
   - Dark background (#0a0a0a, #1a1a1a)
   - Light text (#e0e0e0, #ffffff)
   - Teal accent color (#2ee4bb)

2. **main-light.css** - Light theme (original)
   - White background (#ffffff)
   - Dark text (#000000)
   - Bright teal accent (#49fcd4)

## How to Switch Themes

To switch between themes, simply copy the desired theme file to replace the main CSS file:

### Switch to Light Theme:
```bash
cp assets/css/themes/main-light.css assets/css/main.css
```

### Switch to Dark Theme:
```bash
cp assets/css/themes/main-dark.css assets/css/main.css
```

Or manually copy and paste the contents of your preferred theme into `assets/css/main.css`.
